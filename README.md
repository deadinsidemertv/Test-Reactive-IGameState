# Boot Flow — Test Task (Вариант A)

## Как запустить
1. Открыть проект в Unity 6+
2. Открыть сцену `Assets/Scenes/SampleScene`
3. Нажать Play

## Что происходит
- SplashState — ждёт 1 секунду
- LoadState — прогресс-бар заполняется за 5 шагов по 200мс
- MenuState — появляется кнопка Restart, по нажатию возвращает в LoadState

## Что бы я доделал, будь у меня ещё 2 часа

Скорее всего выбрал бы один из двух путей:

**Путь 1 — углубиться в стек:**
Посидел бы и разобрался как именно работает VContainer внутри — как builder регистрирует зависимости и автоматически резолвит конструкторы. То же самое с UniTask — понять что именно происходит внутри когда мы делаем await, как это ложится на PlayerLoop Unity.

**Путь 2 — улучшить визуал:**
- Lerp для плавного движения прогресс-бара вместо резких скачков
- Fade-in / fade-out при переходах между стейтами (у меня в своём движке это уже реализовано через SpriteBatch)
- Простая анимация появления кнопки
